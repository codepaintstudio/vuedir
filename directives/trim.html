v-trim ​

简介 ​

v-trim 指令用于去除字符串两端的空格。

基本用法 ​

给目标元素添加 v-trim 指令，即可去除字符串两端或中间的空格。

查看代码
vue
<script setup lang="ts">
import { vTrim, vClearable } from '@cp-vuedir/core'
</script>

<template>
  <div class="container">
    <input type="text" v-trim v-clearable placeholder="相聚离开 都有时候 没有什么会永垂不朽" />
  </div>
</template>

<style scoped>
.container,
input {
  width: 100%;
  padding: 10px;
  border: 1px solid var(--vp-c-brand);
  border-radius: 8px;
}
</style>
