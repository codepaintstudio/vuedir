v-desatuate ​

简介 ​

v-desatuate是一个可以一键开启网页灰色素调的指令，常用在一些特殊时节。

使用方法 ​

给任意元素绑定v-desatuate的指令，传入一个类型为boolean的isOn参数即可。

查看代码
vue
<script setup lang="ts">
import { ref } from 'vue'
import { vDesaturate } from '@cp-vuedir/core'

const isOn = ref<boolean>(false)
</script>

<template>
  <a-card
    v-desaturate="{
      isOn
    }"
  >
    <a-switch type="round" v-model="isOn" />
    <p>当前处于：{{ isOn ? '灰色素调' : '正常素调' }}模式</p>
  </a-card>
</template>

API ​

属性名说明类型是否必选默认值
isOn
控制是否开启灰色素调模式
boolean
