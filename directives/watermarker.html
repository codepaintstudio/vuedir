v-watermarker ​

介绍 ​

v-watermarker 指令用于在元素上添加水印效果，以保护内容的版权或标识。

API ​

属性名说明类型是否必选默认值
text水印文本内容string否水印文本
direction水印方向，可选值：'horizontal'（水平）/'vertical'（垂直）/ 'diagonal'（对角线）string否diagonal
fontSize字体大小number否16
fontFamily字体族string否Arial
textColor文字颜色string否#000000
opacity透明度number否0.1
gap水印之间的间距number否100
zIndex水印层级number否1000

注意事项 ​

使用限制

  • 确保容器元素具有明确的宽高，否则水印可能无法正确显示
  • 水印会自动适应容器大小
  • 水印层的 pointer-events 设置为 none，不会影响容器内容的交互

性能优化

  • 使用 Canvas 绘制水印，性能优良
  • 水印图案会被缓存，不会重复绘制
  • 容器大小变化时会自动重新计算水印位置
